# **App Name**: BioLive

## Core Features:

- User Authentication & Profile: Enable users to create new accounts, log in securely, and view their personalized profile, including their content and activities.
- Mixed Content Feed: Display a dynamic, scrolling feed on the main page, blending live streams and short-form video 'reels' for seamless content discovery.
- Live Stream Functionality: Allow users to initiate their own live video broadcasts and provide a robust viewer experience for watching ongoing live streams.
- Reel/Video Upload & Playback: Provide tools for users to upload short video clips (reels) and ensure smooth, high-quality playback of all uploaded content.
- Real-time Live Chat: Integrate a real-time chat system within live streams, allowing viewers to send messages and interact with broadcasters and other viewers.
- AI Content Moderation Tool: Implement an AI-powered tool to automatically detect and filter inappropriate or spammy messages within live stream chats to maintain a safe environment.
- Global Search & Navigation: Include a prominent search bar for finding content and users, complemented by intuitive bottom navigation for easy access to core app sections like reels, lives, and notifications.

## Style Guidelines:

- Primary color: Vibrant Green (#51A329) to evoke growth and vitality, drawing from the 'Bio' and 'Live' aspects of the app name. This green is chosen for its lively yet modern feel.
- Background color: A very subtle off-white with a hint of the primary hue (#F3F4F0). This provides a clean and fresh backdrop that highlights content without being distracting, ensuring readability in a light scheme.
- Accent color: A bright, zesty Lime Green (#C9E830). This analogous color provides a high-contrast accent for interactive elements like buttons and notifications, adding an energetic pop to the design.
- Headline and body text font: 'Inter', a modern grotesque-style sans-serif. Its objective, neutral aesthetic ensures excellent readability across diverse content, from video titles to chat messages.
- Utilize a consistent set of line-art icons that are modern, clean, and minimalist. These icons will represent actions like search, profile, live, reels, notifications, and content upload, ensuring clarity and brand consistency.
- Employ a mobile-first, full-screen layout for the content feed, prioritizing immersive viewing. Implement a clear header for search and profile access, and a fixed bottom navigation bar for quick access to key app functionalities.
- Incorporate subtle, fluid animations for transitions between feed items and when new content loads. Use a gentle pulsing animation for live stream indicators to draw attention without being intrusive.
